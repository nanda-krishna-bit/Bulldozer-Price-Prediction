# Bulldozer Price Prediction

## Overview

This project was completed as part of an IIT Madras Machine Learning Kaggle competition.

The objective is to predict the sale price of used bulldozers using historical auction data.

---

## Dataset

The dataset contains information such as

- Machine age
- Model
- Manufacturer
- Usage
- Sale date

---

## Workflow

- Data Cleaning
- Feature Engineering
- Missing Value Handling
- Encoding
- Model Training
- Model Evaluation
- Hyperparameter Tuning
- Final Submission

---

## Models Used

- CatBoost
- XGBoost
- LightGBM
- Ensemble Learning
